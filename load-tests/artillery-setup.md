# Нагрузочное тестирование с Artillery (JSON отчеты)

Этот документ описывает, как запускать нагрузочные тесты с Artillery и получать отчеты в формате JSON.

## 1. Установка Artillery

Установите Artillery глобально с помощью `npm`.

```bash
npm install -g artillery
```

## 2. Запуск тестов

Скрипты в `package.json` полностью автоматизируют процесс: они запускают веб-приложение, выполняют тесты, генерируют JSON-отчет и останавливают приложение.

### Запуск полного цикла тестов

Эта команда последовательно выполнит все сценарии и для каждого создаст свой JSON-отчет в корневой директории проекта.

```bash
npm run load-test:all
```

### Запуск конкретного теста

Вы можете запустить любой из тестов по отдельности.

```bash
# Тестирование галереи
npm run load-test:gallery

# Тестирование загрузки файлов
npm run load-test:upload

# Тестирование взаимодействий
npm run load-test:interaction
```

## 3. JSON-отчет

По завершении каждого теста в корневой директории будет создан `.json` файл с подробными метриками, например: `artillery_report_gallery_... .json`.

Этот файл содержит все данные о производительности (задержки, RPS, коды ответов и т.д.) и может быть использован для дальнейшего анализа или импорта в другие системы визуализации.
